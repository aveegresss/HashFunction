# Hash function

## Описание
Проект **Hash function** реализует хеширование текста через иттеративный блочный шифр ГОСТ 28147-89   

**Хеширование реализуется по 1 формуле из таблицы:**

## Основные возможности
- Хеширование текста

## Технологии
